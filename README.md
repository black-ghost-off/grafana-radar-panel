1. Install dependencies

   ```bash
   npm install
   ```

2. Build plugin in development mode and run in watch mode

   ```bash
   npm run dev
   ```
3. Run grafana docker container

   ```bash
   sudo docker-compose up -d
   ```
4. Go to 

   ```bash
   http://localhost:3001/
   ```
